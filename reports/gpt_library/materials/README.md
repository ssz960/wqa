# Material intake

- Put new raw captures or link notes in `incoming/` with source URL, capture date,
  source type, intended scope, and user objective.
- Put distilled researcher-ready notes in `distilled/` after value filtering, light
  history/duplicate review, field/operator resolution, and evidence labeling.
- Do not copy generated `material_manager_*` status snapshots here. They are runtime
  history and remain discoverable through `source_inventory_current.csv`.
